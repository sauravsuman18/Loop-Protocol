# Loop Protocol

## Overview

Loop Protocol is a first-person psychological horror game developed in Unity. Players navigate a looping corridor where subtle environmental changes create suspense and drive progression.

---

## Gameplay

- Loop-based exploration
- Environmental storytelling
- Interactive objects
- Trigger-based events
- Horror atmosphere

---
